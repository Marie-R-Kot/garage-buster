# Распознавалка гаражей

>Задеплоено на: [garage-buster.streamlit.app](https://garage-buster.streamlit.app/)

Практический пример задачи детекции в реальной жизни – распознавание незагеристрированных городских построек на примере гаражей. Найденные объекты верифицируются в Росреестре на предмет наличия регистрации объекта.

### Как это работает
- Пользователь вводит адрес или выбирает интересующую его область на карте
- Сервис определяет центр области и запрашивает в Static API (2GIS) тайл с постройками
- Тайл поступает на inference дообученной модели YOLOv8, после обработки получаем мировые координаты найденных объектов
- Координаты отправляются в Росреестр через не очень публичный API, где определяется наличие их регистрации

### О дообучении модели
- Использовался dataset, в котором данные были собраны и обработаны самостоятельно
- Модель идентифицирует объекты, похожие на гаражи: по размеру, форме и расположению относительно других построек

### Стэк
- **Frontend**: Streamlit
- **Картография**: OpenStreetMap map widget / 2GIS Static API
- **ML-модель**: YOLOv8
- **Геокодирование**: OpenStreetMap Nominatim
- **Рег. данные**: Непубличный API Росреестра

### Возможные улучшения
- Больше данных для более точной классификации (и автоматический парсинг таких данных)
- Распознавание фотографий объекта, полученных с 2GIS (доп.критерий типа сооружения)
- Использование платных API Росреестра для предоставлении данных не только о самом факте регистрации, но и необходимых подробностей
